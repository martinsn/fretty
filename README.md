# Fretty 🎸

Guitar fretboard trainer with spaced repetition.

## Features

- Learn guitar notes across the fretboard
- Spaced repetition system (SM-2 algorithm)
- Multiple practice modes
- Progress tracking
- PWA-installable (works offline)

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- PWA (Service Worker + Manifest)

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
